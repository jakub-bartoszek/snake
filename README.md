## Instalation

1. Clone the repository:

   ```
   git clone https://github.com/jakub-bartoszek/snake-game.git
   cd snake-game
   ```

2. Install dependencies:
   ```
   npm install
   ```
3. Run the app:
   ```
   npm start
   ```
